Языки сайта:

1. JavaScript

HTML (HyperText Markup Language) — язык разметки, который определяет структуру веб-страницы. С его помощью создаются заголовки, абзацы, изображения, ссылки, таблицы, формы и другие элементы контента.

2. CSS

CSS (Cascading Style Sheets) — язык стилей, используемый для оформления веб-страниц. Позволяет управлять цветами, шрифтами, размерами, отступами, анимациями и расположением элементов на странице.
